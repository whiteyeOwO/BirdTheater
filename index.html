第一章

都市三俠

甚麼是都市三俠?

讓我們好好自我介紹一下!

斯式繡眼

Zosterops simplex

別名

綠繡眼 / Swinhoe's White-eye / 暗綠繡眼鳥 / 青笛仔

目科別

燕雀目 / 繡眼科 / 繡眼屬

特徵

體長約 11公分，背面大致為黃綠色，有白色眼圈和黑色下眼線。 喙是黑色，喉部和尾下覆羽為黃色，腹部主要是白色，腳為鉛灰色。

簡介

是台灣普遍的留鳥，目前瀕危等級無危。 已適應人類營造的綠地環境，在公園、校園、行道樹、果園、 陽台盆栽等地方皆可發現牠們的蹤跡。不會進入完全的天然林內， 對人類毫不畏懼，已完全演化成與人類共存。成群生活， 鳥群規模約20至50隻，夜晚時數隻會一起依偎在樹上的細枝上睡覺。

甚麼是都市三俠?

接下來是翁董!

白頭翁

Pycnonotus sinensis

別名

白頭鵯 / Light-vented Bulbul / 白頭殼仔

目科別

燕雀目 / 鵯科 / 鵯屬

特徵

體長約19公分，前額、後頸、眼睛四周及兩頰為黑色， 頭後方及尾下覆羽白色，背、中覆羽呈現黃橄欖綠色， 胸部淺棕色。喙與腳黑色。

簡介

是台灣普遍的留鳥，目前瀕危等級無危。 經常出現在中低海拔的次生林、灌木叢、農田、果園、 都市公園與行道樹等環境中。不太怕人。通常為2~5隻成群生活。

甚麼是都市三俠?

最後是花生麻糬 麻雀!

麻雀

Passer montanus

別名

樹麻雀 / Eurasian Tree Sparrow / 家雀 / 厝尾鳥仔 / 粟鳥仔

目科別

燕雀目 / 麻雀科 / 麻雀屬

特徵

體長約13公分，喙黑或褐色，頭上、後頸、背為紅褐色，臉、胸、 腹、尾下覆羽為灰白色，臉頰有大型黑色圓斑。喉部黑色， 背有明顯黑色軸斑，腳呈現淡黃褐色。

簡介

是台灣普遍的留鳥，目前瀕危等級無危。 麻雀是台灣鳥類中最能夠適應人類棲地的鳥種，在都市環境中， 麻雀可說是最普遍的。海拔500m以下的城鎮、鄉村、農田的住家附近經常可見。 喜歡於地面及稻田中覓食，經常數隻或成群活動，群聚性相當明顯。

看完上述介紹後有沒有更了解我們呢?
每年3~8月是我們的繁殖季，那時可以留心傾聽周遭環境的聲音，
會發現我們距離很近!
還有如果我們幸運地造訪陽台育雛，
請遠遠地守護我們就好，不用給予食物和水。

都市裡除了我們，還有其他鄰居，
如果他們出現了會簡單介紹一下!

斑文鳥

別名

黑嘴筆仔 / Scaly-breasted Munia

目科別

燕雀目 / 梅花雀科 / 文鳥屬

特徵

體長約 11公分，喙灰黑色，頭、頸、背、 雙翼、尾羽大致褐色，胸、腹部及尾下覆 羽白色、胸腹帶黑褐色鱗狀斑，腳鉛色。

簡介

棲息平地及低海拔丘陵，喜歡農田及芒草 叢，群聚性相當明顯。飛行時成直線波浪狀 前進。

我們的鄰居很多，大小、習性、外型都很不一樣，

第三章

常見外來種

惡鄰居來了!

不過不是每個都很兇殘?

亞洲輝椋鳥

Aplonis panayensis

別名

菲律賓椋鳥 / Asian Glossy Starling

目科別

燕雀目 / 椋鳥科 / 輝椋鳥屬

特徵

體長約19公分，全身黑綠色，尾方形略 短。腹面淺米色。 頰及喉有細黑縱紋。胸 腹部具粗縱斑。眼血紅色。喙、腳黑色。

簡介

多數棲息於都市內。群居性， 總是在高處 活動，幾乎不落至地面。

看完上面惡鄰居的介紹後，
發現不是每個惡鄰居都是像8+9一樣，
可能有些還有點單純(?
但是他們最可怕的地方不是他們的個性，
而是他們強大的適應環境能力。
因為如此，他們的族群數量逐年快速上升，
讓我們原生種的居住空間、食物都被壓縮，
大大減少了我們生存的機會。

除了惡鄰居的侵擾，其實還有另一個更大的威脅...

尾聲

窗殺

這個巨大威脅就是......窗殺!

窗殺是都市裡常常造成鳥類死亡的原因，原因有二，
第一個是光滑的亮面反射天空周遭樹景形成鏡面效果，
第二個則是透明的玻璃造成可以通過的錯覺。
以上兩點都是因為窗戶造成的假象讓鳥類誤會而死亡。

那 我們如何預防窗殺呢?

很簡單，只要讓窗戶不那麼通透，
讓鳥可以辨識出來就好!

改善玻璃材質(霧面、雕刻裝飾...)

掛上垂墜物(裝飾、盆栽...)

貼上大面積貼紙(需要幾乎貼滿)

版權

總而言之，都市有許多種野鳥，
有本土的、外來的，或是短暫停留的過客。
雖然不是每一種的都是友善的，但他們也只是被人類引進後被棄養，必須在新環境努力生存。
還有他們都會遇到的窗殺問題，
這是人類造成且只有人類可以改善的，
因此我們應該加強宣導窗殺防治。

最後，謝謝看到這裡的大家!
希望未來都能為守護生態盡一份力!
